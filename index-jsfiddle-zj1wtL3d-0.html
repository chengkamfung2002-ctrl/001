The World Wide Web is a massive network of interconnected digital files stored on computers all over the world. When you open a browser like Chrome or Safari, you are using a tool to read those files.

Lesson 2: What is HTML?

HTML stands for HyperText Markup Language. It acts as the structural skeleton of every single web page on the internet.

Think of it like building a house: HTML represents the wood framing and beams. Without it, text, images, and buttons wouldn't know where to sit on the screen!

📖 第一课：构建互联网世界的基石 (HTML/CSS)

如果说互联网是一座摩天大楼，那么 HTML 就是大楼的钢筋混泥土框架。它规定了哪里是“门”（按钮），哪里是“窗”（图片），哪里是“墙壁”（文本段落）。

而 CSS 则是大楼的室内设计师与灯光师。通过 CSS，我们可以调整字体颜色、给卡片加上柔和的阴影、甚至设计绚丽的渐变色底图（就像你现在看到的这个现代化管理系统一样）。

⚡ 第二课：让网页拥有灵魂的魔法 (JavaScript)

有了骨架（HTML）和衣服（CSS），网页依然是静止的。如果想要网页对用户的点击做出反应、如果需要实时计算复杂的数学公式，我们就必须引入 JavaScript (JS)。

JavaScript 赋予了网页“思考”与“动作”的能力。例如，你现在点击下方按钮时，进度条之所以会自动向前平滑推移、积分在悄悄上涨，这就是后台 JavaScript 正在疯狂运转的结果！
